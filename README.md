Conditional Utility
===================

Usage
-----

### CommonJS

~~~ js
const {hasValue} = require('@taufik-nurrohman/has');

console.log(hasValue('foo', ['foo', 'bar', 'baz']));
~~~

### ECMAScript

~~~ js
import {hasValue} from '@taufik-nurrohman/has';

console.log(hasValue('foo', ['foo', 'bar', 'baz']));
~~~
